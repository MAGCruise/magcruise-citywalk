早稲田祭ウォークラリー (仮)

Navi + Checkin + Quiz

早稲田祭をめぐり，早稲田大学のランドマークや企画イベントにチェックイン．クイズを解いて早稲田文化を体感するウェブアプリ

ウォークラリーの流れ

  1. ユーザ登録をします．
  2. チェックポイントを選び，ナビに従って，チェックポイントへ行きます．
  3. チェックポイントで「合い言葉」を入力してチェックインし，クイズに答えます．
  4. 新たなチェックポイントを目指します．
  5. ランキング画面で表示される総ポイント数が50pt以上になったらインフォメーションにお越し下さい．

注意事項

対応Webブラウザは，Safari または Google Chrome (iOS)，Google Chrome (Android)です． また，GPSとインターネット接続を必要とします．

このアプリでは，ナビ画面を開いている間，GPSログが記録されサーバに送信されます． GPSログは，ニックネームとともに保存されますが，論文発表などの学術目的以外には利用しません．
