はじめに

ワセナビ

Navi + Checkin + Quiz

早稲田祭をめぐり，ランドマークやイベントにチェックイン．クイズを解いて早稲田文化を体感しよう．

ワセナビの流れ

  1. ユーザ登録をします．
  2. チェックポイントを選び，ナビに従って，チェックポイントへ行きます．
  3. チェックポイントで「合い言葉」を入力してチェックインし，クイズに答えます．
  4. 新たなチェックポイントを目指します．
  5. ランキング画面で表示される獲得ポイントが50pt以上になったらインフォメーションにお越し下さい．

注意事項

対応Webブラウザは，Safari または Google Chrome (iOS)，Google Chrome (Android)です．また，GPSとインターネット接続を必要とします．

このアプリでは，ナビ画面を開いている間，GPSログが記録されサーバに送信されます． GPSログは，ニックネームとともに保存されますが，論文発表などの学術目的以外には利用されません．

移動するときは周辺の状況に十分注意してご利用下さい．
