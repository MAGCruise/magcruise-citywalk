はじめに

ワセナビ

Navi + Checkin + Quiz

早稲田祭をめぐり，ランドマークやイベントにチェックイン．クイズを解いて早稲田文化を体感しよう．

推奨環境

GPSとインターネット接続が使える以下の環境を推奨します．

  • iOS (iPhone, iPad, Macなど)
    • Safari (最新版)
    • Google Chrome (最新版)
  • Android
    • Google Chrome (最新版)
  • Windows
    • Google Chrome (最新版)
注意事項
  • このアプリをお使いになる場合，ナビ画面を開いている間，GPSログが記録されサーバに送信されることに同意下さい． GPSログは，ニックネームとともに保存されますが，論文発表などの学術目的以外には利用されません．
  • このアプリにGPSの利用を許可する場合，以下のようなアラートが表示されたときに，許可を押してください．
