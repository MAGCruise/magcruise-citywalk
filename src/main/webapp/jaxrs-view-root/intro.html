はじめに

ワセナビ

Navi + Checkin + Quiz

早稲田祭をめぐり，ランドマークやイベントにチェックイン．クイズを解いて早稲田文化を体感しよう．

推奨環境

GPSとインターネット接続が使える以下の環境．

  • iOS : Safari または Google Chrome (最新版)
  • Android : Google Chrome (最新版)
  • Windows : Google Chrome (最新版)
注意事項
  • 本アプリをお使いになる場合，ナビ画面を開いている間，GPSログが記録されサーバに送信されることに同意下さい． GPSログは，ニックネームとともに保存されますが，論文発表などの学術目的以外には利用されません．
  • 本アプリにGPSの利用を許可する場合，以下のアラートが表示されたときに許可を押して下さい．

移動中は周辺に十分注意してご利用下さい．
