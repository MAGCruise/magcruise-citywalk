推奨環境

GPSとインターネット接続が使える以下の環境

  • iOS : Safari または Google Chrome (最新版)
  • Android : Google Chrome (最新版)
  • Windows : Google Chrome (最新版)
注意事項
  • 本アプリに端末の緯度経度情報の利用を許可する場合，表示されるアラートで許可を選択して下さい．
  • 位置情報の利用許可を与えた場合，端末の緯度経度情報がサーバに送信され記録されることに同意下さい．この情報はニックネームとともに保存されますが，論文発表などの学術目的以外には利用されません．

移動中は周辺に十分注意してご利用下さい．
